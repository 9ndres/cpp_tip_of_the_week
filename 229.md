* **Did you know about python's named tuples?**

</p></details><details open><summary>Example</summary><p>

```python
from collections import namedtuple

nt = namedtuple("price", "size")

nt.price = 42;
nt.size = 100;

assert 42 == nt.price and 100 == nt.size
```
